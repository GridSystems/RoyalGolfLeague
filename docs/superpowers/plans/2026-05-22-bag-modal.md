# Bag Modal Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Move "My Bag" out of the Players tab bottom panel and into a modal opened by a dedicated button.

**Architecture:** Remove `#bagSetupPanel` from `#view-players`, add it inside a new `bagModal` (same `modal-bg` + `modal modal-sm` pattern as all other modals), add a `🎒 My Bag` button in the Club Roster panel header that shows only when a player is active, and add `openBagModal()` that sets the modal title and calls `renderBagSetup()` before showing it.

**Tech Stack:** Vanilla JS, single `index.html`.

```js
function openBagModal(){
  const p=players.find(x=>x.id===activeId);
  if(!p){alert('Select your player first.');return;}
  document.getElementById('bagModalTitle').textContent=`🎒 ${p.name}'s Bag`;

}
```
